Adds animation and visibility options to numerous WordPress blocks

== Description ==

This plugin contains some of the functionality for the PRO Version of the Flat Blocks WordPress Theme. These are provided as a plugin because a plugin is the most suitable approach to these features. In addition, this means you can use this plugin and have these features even on a site that uses a different theme.

1. It adds an animation control to groups, columns, buttons, etc. to allow for various fade-in and hover animations.

2. It adds a visibility control to groups to allow displaying or hiding them on mobile, tablet, or desktop.

== Installation ==

To install this plugin:

1. Upload the plugin files to the `/wp-content/plugins/pro-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

== Developer Notes ==

Enter the directory of this plugin and then you can run the following commands:

  $ npm start
	Builds the code for development and continues to watch for changes to source code

  $ npm run build
    Builds the code for production

  $ npm run format
    Formats files

  $ npm run lint:css
    Lints CSS files

  $ npm run lint:js
    Lints JavaScript files

  $ npm run plugin-zip
    Creates a zip file for a WordPress plugin

  $ npm run packages-update
    Updates WordPress packages to the latest version
